Руководство по настройке GitHub

Пошаговые инструкции для начала работы с Git и GitHub

  • Регистрация
  • Локальный Git
  • SSH ключ
  • VS Code

Как зарегистрироваться на GitHub

  1. Перейдите на github.com
  2. Нажмите кнопку "Sign up"
  3. Заполните форму регистрации:
    • Введите email
    • Создайте пароль
    • Выберите имя пользователя
  4. Подтвердите email
  5. Настройте свой профиль

Настройка локального Git

  1. Скачайте Git с официального сайта: git-scm.com
  2. Установите Git, следуя инструкциям установщика
  3. Откройте терминал и выполните базовую настройку:
    git config --global user.name "Ваше имя" git config --global user.email "ваш@email.com"
  4. Проверьте установку:
    git --version

Генерация SSH ключа

  1. Откройте терминал
  2. Сгенерируйте SSH ключ:
    ssh-keygen -t ed25519 -C "ваш@email.com"
  3. Нажмите Enter для сохранения ключа в папке по умолчанию
  4. Добавьте ключ в ssh-agent:
    eval "$(ssh-agent -s)" ssh-add ~/.ssh/id_ed25519
  5. Скопируйте публичный ключ:
    cat ~/.ssh/id_ed25519.pub
  6. Добавьте ключ в настройках GitHub (Settings → SSH keys)

Настройка VS Code

  1. Установите VS Code с официального сайта
  2. Установите расширение "GitHub Pull Requests and Issues":
    • Откройте VS Code
    • Нажмите Ctrl+Shift+X
    • Найдите и установите расширение
  3. Войдите в GitHub через VS Code:
    • Нажмите на иконку GitHub в боковой панели
    • Нажмите "Sign in to GitHub"
    • Следуйте инструкциям в браузере
